Q2

  • P: Qual a correspondência entre o padrão MVC e as partes de um sistema de informação desenvolvido com o framework JSF?
  • R:
    • O papel do model, com as regras de negócio da aplicação, é desempenhado pelo conjunto de classes de domínio;
    • O papel da view é desempenhado pelo conjunto de páginas XHTML denominadas facelets;
    • O papel do controller é desempenhado pelo conjunto de classes Java denominadas "managed beans".
  • P: Por que no ciclo de vida de uma aplicação JSF é executada a fase de processamento de validações antes da atualização dos valores do modelo?
  • R: Para garantir que a atualização dos valores do modelo ocorra sem problemas, pois os dados dos componentes já terão sido validados.
